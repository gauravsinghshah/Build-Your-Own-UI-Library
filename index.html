The lightweight, utility-first CSS framework for rapid and elegant web development.

"MythicalUI is a powerful, extensible, and modern frontend framework. Build sleek, responsive interfaces with our utility-first classes, modular components, and Sass-powered customization. Bring your ideas to life faster with lightweight plugins and a flexible grid system."

npm install
