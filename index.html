Position

static

static is the default position for HTML elements. They appear in the normal document flow and aren't affected by top, left, bottom, or right.


<div class="position-static">
  Static position (default, no offset)
</div>

relative

relative positions the element relative to its normal position. You can offset it using top, left, etc.


<div class="position-relative">
  Relative position
</div>

absolute

absolute positions the element relative to the nearest ancestor with a positioned context. It is removed from the normal document flow.


<div class="position-relative">
  <div class="position-absolute">
    Absolute inside relative
  </div>
</div>

fixed

fixed positions the element relative to the viewport. It remains in the same place even during scroll.


<div class="position-fixed">
  Fixed element
</div>

sticky

sticky acts like relative until the element crosses a certain scroll threshold, then it behaves like fixed.


<div class="position-sticky">
  Sticky element
</div>
